 Passenger Counter App 🚂

A simple and elegant web application to track the number of passengers entering a station. This app features a beautiful station background and provides an easy way to increment counts and save entries.

 Features ✨

- **Increment Button**: Quickly increment the passenger count
- **Save Button**: Save the current count and maintain a history of previous entries
- **Persistent History**: View all previous entry counts in one place
- **Responsive Design**: Clean, centered layout with a professional appearance
- **Background Theme**: Beautiful station-themed background image

 Screenshots 📸

The app displays:
- A large, easy-to-read passenger count
- Two prominent buttons for incrementing and saving
- A history section showing all previous entries

 How to Use 🚀

1. Open `index.html` in your web browser
2. Click the **INCREMENT** button to increase the passenger count
3. Click the **SAVE** button to record the current count
4. View your counting history under "Previous entries"

 Files 📁

- `index.html` - Main HTML structure
- `index.css` - Styling and layout
- `index.js` - JavaScript functionality
- `station.jpg` - Background image

 Technologies Used 💻

- **HTML5** - Structure and markup
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactive functionality
